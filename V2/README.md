# Override Settings

Tools > Manage filesystem <br>
Criar o ficheiro config.txt

```js
HanDelayWait 1000
HanDelayError 35000
HanTimeout 2000
HanRestart 0
TelePeriod 0
```

O ficheiro só é aplicado aos 30 segundos depois de iniciado.
